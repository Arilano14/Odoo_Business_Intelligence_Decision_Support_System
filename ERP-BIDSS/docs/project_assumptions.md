# Project Assumptions

- **Assumption 1:** Odoo data (Sales, Inventory, Purchase, Accounting) tersedia.
- **Assumption 2:** Dataset Kaggle dapat digunakan untuk volume data.
- **Assumption 3:** Forecast menggunakan data historis tanpa memperhitungkan krisis ekonomi global yang tak terduga.
- **Assumption 4:** Power BI menggunakan Import Mode.
- **Assumption 5:** User hanya membaca dashboard (Read-Only).
