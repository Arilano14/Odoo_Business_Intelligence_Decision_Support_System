# Use Cases

- **UC1:** CEO melihat company performance secara menyeluruh.
- **UC2:** Inventory manager mengecek reorder recommendation berdasarkan perhitungan EOQ/ROP.
- **UC3:** Sales manager melihat forecasting penjualan menggunakan Prophet/SARIMAX.
- **UC4:** Procurement manager melihat supplier ranking untuk mitigasi risiko.
- **UC5:** Finance manager menganalisis rasio cashflow dan AR/AP days.
