# Stakeholder Analysis

| Stakeholder | Need |
| :--- | :--- |
| **CEO** | Company performance, profitability, business health score. |
| **Sales Manager** | Sales trends, revenue forecasting, top products. |
| **Inventory Manager** | Stock optimization, overstock/stockout alerts, ROP. |
| **Procurement Manager** | Supplier evaluation, supplier risk score, lead time tracking. |
| **Finance Manager** | Revenue analysis, cashflow risk, expense tracking. |
| **Business Analyst** | KPI monitoring, data quality tracking, dashboard reliability. |
